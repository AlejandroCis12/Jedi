# Jedi - JD Pipe Sales

## Project Overview
**Jedi (JD Pipe Sales)** is a specialized web platform designed for the commercialization and technical cataloging of industrial piping, valves, and high-pressure connections. This project was developed to provide a streamlined, high-performance interface for B2B procurement in the hydraulic and industrial sectors.

Live Demo: [https://alejandrocis12.github.io/Jedi/](https://alejandrocis12.github.io/Jedi/)

## Mathematical & Computational Implementation
As an **Applied Mathematician**, I utilized this project to implement structured information systems and efficient UI logic:

* **Categorical Data Mapping:** Developed a logical framework to organize diverse industrial products (Galvanized Steel, Bronze, Industrial Valves) into searchable and intuitive datasets.
* **Efficient UI Dynamics:** Implemented custom JavaScript for interactive elements, such as the brand carousel (ARES, RUGO, ALFA, etc.), focusing on minimal resource consumption and smooth execution.
* **Industrial Taxonomy:** Designed the information architecture to align with international technical standards (ASTM, NOM, ASME), ensuring that the digital structure reflects physical engineering requirements.

## Tech Stack
* **Frontend:** Semantic HTML5, CSS3 (Advanced Layouts).
* **Interactivity:** Vanilla JavaScript (ES6+).
* **Integration:** Strategic linking with the corporate parent site (Hidrosistemas) to create a cohesive digital ecosystem.

## Key Features
* **Technical Cataloging:** Specific sections for certified piping and specialized valves.
* **Performance Optimization:** Fast-loading interface designed for industrial environments with limited connectivity.
* **Responsive Architecture:** Fully adapted for mobile and desktop use to facilitate on-site technical consultations.

## Academic Relevance
This project showcases the practical application of **Set Theory and Logical Classification** in a commercial environment. It demonstrates proficiency in translating complex industry standards into efficient computational structures, a core requirement for Advanced Computing research.
